ooo2pdf
-------
command line tool driving OOo for PDF export

installation:
    install the ooo2pdf.oxt OOo extension (Tools > Options > Extension Manager)
    Verify the path of ooo2pdf script (should be ok)

usage:
$ ooo2pdf <list of file to convert>

the resulting files are in the original place and named with .pdf extensions
